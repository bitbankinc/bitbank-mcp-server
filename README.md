# bitbank MCP Server

This project is a Model Context Protocol (MCP) server for [bitbank](https://bitbank.cc/).

## Tools

- get_ticker
  - Get ticker data for a trading pair
  - input:
    - `pair` (string): Trading pair to get ticker data for. eg. btc_jpy, eth_jpy
